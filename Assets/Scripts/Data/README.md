# Example ScriptableObject Assets

This folder contains example items and recipes to get you started with the game.

## Creating Your Own Assets

### To create a new Resource Item:
1. Right-click in this folder
2. Choose "Create > My Place RPG > Items > Resource"
3. Name your item (e.g., "Wood", "Stone", "Iron Ore")
4. Fill in the properties in the Inspector

### To create a new Crafting Recipe:
1. Right-click in this folder  
2. Choose "Create > My Place RPG > Crafting > Recipe"
3. Name your recipe (e.g., "Wooden Plank", "Stone Block")
4. Set up the required items and result item

### To create a basic Item:
1. Right-click in this folder
2. Choose "Create > My Place RPG > Items > Base Item"
3. Configure as needed for tools, consumables, etc.

## Tips for Asset Creation

- Give items clear, descriptive names
- Set appropriate values based on rarity and usefulness
- Use consistent naming conventions
- Test your recipes in-game to make sure they're balanced
- Create items in logical progression (basic → advanced)

## Example Progression

**Basic Resources:**
- Wood (from trees)
- Stone (from rocks)
- Plant Fiber (from bushes)

**Processed Materials:**
- Wooden Plank (2 Wood)
- Stone Block (3 Stone)
- Rope (3 Plant Fiber)

**Tools:**
- Basic Axe (2 Wood + 1 Stone)
- Basic Pickaxe (1 Wood + 2 Stone)

**Advanced Items:**
- Wooden Chair (4 Wooden Plank)
- Stone Foundation (8 Stone Block)

Remember: Start simple and add complexity as you learn!
